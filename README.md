# Premium Vale Mobile App

Bu repo, Premium Vale projesinin mobil uygulamasını içerir.

## Kurulum

Mac bilgisayarda kurulum ve çalıştırma talimatları için lütfen [MAC_KURULUM.md](MAC_KURULUM.md) dosyasını inceleyin.

## Proje Hakkında

Bu proje React Native (Expo) ile geliştirilmiştir.
Backend servisi olarak canlı sunucu (`https://eraslan.pythonanywhere.com`) kullanılmaktadır.
